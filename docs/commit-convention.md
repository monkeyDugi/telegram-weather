# 커밋 컨벤션

## 형식
```
타입: 한국어 설명
```

## 타입
| 타입 | 의미 | 예시 |
|---|---|---|
| `feat` | 새 기능 | `feat: GeekNews 스크래퍼 구현` |
| `fix` | 버그 수정 | `fix: 포인트 파싱 오류 수정` |
| `docs` | 문서 변경 | `docs: CLAUDE.md 초기 설정` |
| `refactor` | 코드 개선 | `refactor: 메인 함수 분리` |
| `chore` | 기타 잡일 | `chore: requirements.txt 추가` |

## 규칙
- 커밋 단위: 기능 하나 완료될 때마다
- 커밋 메시지 언어: 한국어
- Claude가 기능 완료 후 커밋 메시지를 제안하고, 사용자 승인 후 커밋
